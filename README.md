# Texas Salary Prediction Project

## Project Overview
This project predicts employee salaries using Machine Learning techniques.

## Dataset
The dataset contains employee demographic, education, experience, and salary-related information.

## Project Workflow
1. Data Understanding
2. Exploratory Data Analysis (EDA)
3. Data Preprocessing
4. Feature Engineering
5. Model Building
   - Linear Regression
   - Decision Tree Regressor
   - Random Forest Regressor
   - Gradient Boosting Regressor
   - XGBoost Regressor
6. Model Evaluation
7. Conclusion
