# Texas Salary Prediction Project

## Project Overview

This project focuses on predicting employee salaries in Texas using Machine Learning techniques. The objective is to analyze the impact of demographic, educational, and professional factors on salary levels and develop predictive models capable of estimating employee compensation accurately.

The project follows a complete end-to-end Data Science workflow, including data cleaning, exploratory data analysis (EDA), feature engineering, data preprocessing, model development, performance evaluation, and business interpretation of results.

---

## Business Problem

Organizations often need reliable salary estimation models to support workforce planning, compensation analysis, budgeting, and talent acquisition strategies. This project aims to build predictive models that can accurately estimate employee salaries based on available employee attributes.

---

## Dataset Information

The dataset contains employee-related information such as:

- Education Level
- Years of Experience
- Age
- Gender
- Job Role
- Other demographic and professional attributes

**Target Variable:** Salary

## Project Workflow

### 1. Data Understanding
- Dataset inspection
- Data types analysis
- Missing value identification
- Duplicate value checking

### 2. Exploratory Data Analysis (EDA)
- Univariate Analysis
- Bivariate Analysis
- Distribution Analysis
- Correlation Analysis
- Outlier Detection

### 3. Data Preprocessing
- Missing value handling
- Feature encoding
- Feature scaling
- Data transformation

### 4. Feature Engineering
- Creation of meaningful features
- Feature selection
- Data preparation for machine learning models

### 5. Machine Learning Models
The following regression models were developed and evaluated:

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Gradient Boosting Regressor
- XGBoost Regressor

### 6. Model Evaluation
Models were compared using:

- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

### 7. Model Comparison & Selection
Performance comparison was conducted to identify the most accurate model for salary prediction.
